# Waveform Marketing — Brand Context for Claude

## Business
Waveform Marketing is a boutique marketing consultancy run by Kim Nelson. It serves boutique fitness studio owners on the South Shore of MA, replacing poor DIY marketing with professional growth marketing, website management, and marketing operations.

## Visual Identity

### Color Palette
| Name | Hex | Usage |
|------|-----|-------|
| Navy | `#1A2535` | Primary headings, table headers, title |
| Slate | `#3A5A7A` | Subheadings, section labels |
| Granite | `#8A8880` | Notes, captions, footer, italic callouts |
| Dark | `#222222` | Body text |
| White | `#FFFFFF` | Table header text |

### Typography
| Element | Size | Weight | Color |
|---------|------|--------|-------|
| Document title | 22pt | Bold | Navy |
| H1 (sections) | 15pt | Bold | Navy + bottom border |
| H2 (subsections) | 11pt | Bold | Slate |
| Body | 10pt | Regular | Dark |
| Notes/captions | 10pt | Italic | Granite |
| Table headers | 9pt | Bold | White on Navy |
| Table body | 9pt | Regular | Dark |

### Document Style
- Clean, professional, no-jargon
- Alternating table row colors: `#F5F3F0` / `#FFFFFF`
- 1" margins, 8.5×11 page
- Section headers have a Navy bottom border rule

## Voice & Tone
- Direct and confident — no hedging
- Scope-based thinking, not hours-based
- Plain language for small business owners
- Professional but not corporate
